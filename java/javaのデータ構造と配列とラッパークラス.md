# javaのデータ構造

## 配列の生成
```
  データ型[] 配列の名前 = new データ型[配列のサイズ];
```

## リストの生成
```
import java.util.*;
List<データ型> リストの名前 = new ArrayList<データ型>();

List list = new ArrayList();

```

## リストにデータを追加

リストの名前.add( データ );
正し List.ofで用意したListには追加できない。そこで以下のように変更、追加できるListを使用する
var a = new ArrayList<String>();
